## 📌 주의사항

1. **데모 환경**: 모든 데이터는 Mock이며 실제 서버 연동 없음
2. **보안 기능**: 웹 환경에서만 완전히 동작
3. **결제**: UI만 구현, 실제 결제 처리 없음
4. **Firebase**: 코드는 준비되어 있으나 비활성화 상태
5. **성능**: Mock 데이터로 인한 일부 지연 가능
